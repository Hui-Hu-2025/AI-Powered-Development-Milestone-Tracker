我想开发一个追踪有特殊情况孩子的发育里程碑的工具，基本功能包括：

1. 输入孩子的特殊情况信息 

2. 输入孩子的身高，体重，头围，大运动，语言，精细动作，睡眠，饮食等方面的进展信息，
API能利用大语言模型判断孩子目前的情况是良性发育还是倒退

3. 预测孩子的下一个发育里程碑及时间节点 

4. 支持图片，视频，文字的输入格式。

项目的技术架构如下：

React 前端
      |
      v
 FastAPI 后端
      |
      v
ChromaDB / SQLite 向量数据库
      |
      v
 LLM（OpenAI ）

后端：

conda activate rag_demo_py311

cd C:\Users\huihu\Desktop\AI vibing\Child Development Monitoring Tool\backend
python main.py

前端： 

cd C:\Users\huihu\Desktop\AI vibing\Child Development Monitoring Tool\frontend
npm start
